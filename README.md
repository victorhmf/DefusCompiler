# DefusCompiler

## escopo
Técnicas abordadas:
- Comentários
- Inicialização de Variáveis
- Variáveis não utilizadas
- Nomes significativos de variáveis
- Indentação

Escopo linguagem C:
- Atribuições
- Operações
- Bibliotecas
- If/Else
- For
- Funções
